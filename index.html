Month 02 — July Systems & Data Weeks 5–8 · OS + Networks + Architecture + SQL
Wk 05Operating systems — processes, memory, scheduling ~18 hrs ↓
Day by day
MonProcesses vs threads — states, PCB, context switching
TueCPU scheduling: FCFS, SJF, Round Robin, Priority
WedMemory management: paging, segmentation, fragmentation
ThuVirtual memory, page replacement algorithms (LRU, FIFO)
FriDeadlocks: Coffman conditions, prevention, detection
Weekend goal
Set up WSL2 or a Linux VM. Practise ls, cd, grep, ps, top, chmod, cron — 30 mins of shell daily from now on.
Wk 06Networking — OSI model, TCP/IP, DNS, HTTP/S ~18 hrs ↓
Day by day
MonOSI 7-layer model — each layer, protocols, purpose
TueTCP vs UDP, IP addressing, subnetting, CIDR notation
WedDNS, DHCP, ARP, routing protocols (OSPF, BGP basics)
ThuHTTP/HTTPS, TLS handshake, certificates, REST APIs
FriNetworkChuck: subnetting and VLAN video series
Weekend goal
Use Wireshark to capture and analyse real packets. Reading live traffic cements theory far faster than passive notes.
Wk 07Computer architecture — logic gates to CPU ~18 hrs ↓
Day by day
MonNand2Tetris Part 1: Boolean logic, basic gates
TueNand2Tetris: ALU design, adders, multiplexors
WedMemory chips, flip-flops, registers, RAM architecture
ThuFetch-decode-execute cycle, instruction sets (ISA)
FriPipelining, cache hierarchy, CISC vs RISC comparison
Weekend goal
Complete Nand2Tetris Project 1 — build all basic logic gates in the HDL simulator. Hands-on is the whole point.
Wk 08Databases — SQL, normalisation, ACID + July checkpoint ~18 hrs ↓
Day by day
MonSQL: SELECT, WHERE, JOIN types, GROUP BY, subqueries
TueER modelling, primary keys, foreign keys, relationships
WedNormalisation: 1NF, 2NF, 3NF, BCNF — with examples
ThuACID properties — atomicity, consistency, isolation, durability
FriConcurrency: locks, isolation levels, deadlocks in databases
Month 2 checkpoint
Design and build a PostgreSQL database (e.g. a library system). Write 10 different queries. Stuck? Revise weeks 5–8.
